## 使用方式

### 1. 部署 GitHub 项目

```bash
# 1. 登录获取 token
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"password123"}'

# 2. 创建部署
curl -X POST http://localhost:8080/api/v1/deployments \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "server_id": "YOUR_SERVER_ID",
    "repo_url": "https://github.com/vercel/next.js"
  }'
```

### 2. 部署教程（百度/CSDN/掘金等）

```bash
curl -X POST http://localhost:8080/api/v1/deployments \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "server_id": "YOUR_SERVER_ID",
    "tutorial_url": "https://juejin.cn/post/xxxx"
  }'
```

### 3. 先解析内容再部署

```bash
# 解析 GitHub 项目
curl -X POST http://localhost:8080/api/v1/parse-content \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"repo_url": "https://github.com/xxx/yyy"}'

# 解析教程
curl -X POST http://localhost:8080/api/v1/parse-content \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"tutorial_url": "https://blog.csdn.net/xxx"}'
```

## 启动 API 服务器（需要安装 Go）

由于当前环境未安装 Go，需要：

### 方式一：安装 Go 后启动

```bash
# 安装 Go 1.21
wget https://go.dev/dl/go1.21.6.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.21.6.linux-amd64.tar.gz
export PATH=$PATH:/usr/local/go/bin

# 启动 API
cd /root/aixm
go mod download
go run cmd/server/main.go
```

### 方式二：使用 Docker

```bash
# 构建并运行
docker build -t servermind .
docker run -p 8080:8080 --network host \
  -e DB_HOST=localhost \
  -e DB_PASSWORD=servermind_dev \
  servermind
```
